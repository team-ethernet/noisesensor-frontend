API

Welcome to the API for the database of Noisy things

The API is accessed through /data and is returned as a JSON file. All data is returned as an array with the following fields:

  • bn: the base name of the sensor
  • u: the unit of the measurement
  • v: the decibel value measured
  • t: the unix timestamp of the measurement
A selection of sensor data can be specified by typing a list of the following parameters in the URL as a query string. Parameter order does not matter.

ids

Default value: All sensor id:s

Input format:
ids=id0,id1,...
or
ids=id0&ids=id1&ids=...

Description:
Returns only data from the specified sensors.

startDate

Default value: 1970-01-01_00:00

Input format:
startDate=xxxxxxxxxxxxx (unix timestamp)

Description:
Returns data from the start date and forward.

endDate

Default value: Current date

Input format:
endDate=xxxxxxxxxxxxx (unix timestamp)

Description:
Returns data up until the end date.

sortBy

Default value: Sorted by date

Input format:
sortBy=unixTime, sortBy=name or sortBy=value

Description:
Returns sorted values in given order.

sortOrder

Default: Ascending

Input format:
sortOrder=desc
or
sortOrder=asc

Description: Returns sorted values in ascending or descending order.

average

Default value: False

Input format:
average=true
or
average=false

Description:
Returns average noise level for each sensor, and a global average for all sensors.

minNoiseLevel

Default value: 0

Input format:
minNoiseLevel=x

Description:
Returns data where the noise level is above minNoiseLevel

maxNoiseLevel

Default value: 200

Input format:
maxNoiseLevel=x

Description:
Returns data where the noise level is below maxNoiseLevel
